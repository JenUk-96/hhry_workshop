# Курсовой проект для онлайн школы skypro 
## Описание проекта:
+ Программа получает информацию о вакансиях с платформы hh.ru в России, 
+ Позволяет сохранять информацию в файл и удобно с ней работать:
+ Добавлять,
+ Удалять,
+ Фильтровать
